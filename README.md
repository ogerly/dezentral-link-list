# IPFS - InterPlanetary File System
IPFS - IPFS (InterPlanetary File System) ist ein Peer-to-Peer-Protokoll zur Verteilung von Dateien. 
Stellen Sie es sich als ein Dateisystem vor, das die Ideen hinter bittorrent und git nutzt, bei dem 
Daten inhaltsadressierbar und unveränderlich sind.
https://ipfs.io/

IPFS Dokumentation : https://docs.ipfs.io/

InterPlanetary File System: https://en.wikipedia.org/wiki/InterPlanetary_File_System
